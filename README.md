# python-ecommerce-data-analysis
End-to-end e-commerce data analysis project using Python, Pandas, Matplotlib, and Seaborn to analyze 50,000 transactions, identify revenue trends, customer behavior, and generate business insights through visualizations and feature engineering.
# E-Commerce Data Analysis with Python

## Project Overview

This project analyzes an e-commerce transactions dataset containing **50,000 transactions** to uncover business insights related to customer behavior, product performance, payment preferences, and revenue trends.

The project demonstrates an end-to-end data analytics workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, and data visualization using Python.

---

## Dataset Information

* **Records:** 50,000 transactions
* **Features:** 8 original columns
* **Tools Used:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

### Dataset Columns

* Transaction_ID
* User_Name
* Age
* Country
* Product_Category
* Purchase_Amount
* Payment_Method
* Transaction_Date

---

## Project Objectives

The project aims to answer the following business questions:

1. How many transactions occurred?
2. What is the total revenue generated?
3. Which countries generate the most revenue?
4. Which product categories perform best?
5. Which payment methods are preferred by customers?
6. Who are the highest-value customers?
7. What are the spending patterns of customers?

## Exploratory Data Analysis (EDA)

The following analyses were conducted:

### Revenue Analysis

* Total Revenue: **R25,157,989.65**
* Average Purchase Amount: **R503.16**
* Highest Transaction Value: **R999.98**
* Lowest Transaction Value: **R5.04**

### Customer Analysis

* Total Transactions: **50,000**
* Top Customers identified by total purchase amount
* Customer spending segmented into:

  * High Spenders: 25,165
  * Medium Spenders: 15,077
  * Low Spenders: 9,758

### Product Analysis

* Highest Revenue Category: **Sports**
* Most Purchased Category: **Toys**

### Payment Analysis

* Most Preferred Payment Method: **UPI**

### Geographic Analysis

* Highest Revenue Country: **France**

## Key Insights

* The business generated over **R25 million** in revenue.
* The average customer purchase was approximately **R503**.
* Sports products generated the highest revenue.
* Toys had the highest number of purchases.
* UPI was the most preferred payment method.
* France generated the highest revenue among all countries.
* More than half of all transactions came from high-spending customers.
